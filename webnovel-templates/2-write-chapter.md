# Write Next Chapter with Context

## Purpose
Generate the next chapter of your webnovel using the story bible and previous chapter context managed through Supermemory.

## Prerequisites
- Story bible exists (created via template 1)
- Previous chapters written (if not chapter 1)
- Supermemory context manager configured

## Instructions

### 1. Context Retrieval
Using Supermemory, retrieve:
- **Story Bible**: Core world rules, characters, plot structure
- **Recent Chapters**: Last 2-3 chapters for continuity
- **Character States**: Current location, relationships, emotional state
- **Plot Position**: Where we are in the current arc
- **Foreshadowing**: Any planted seeds that need attention

- **Opening Hook**: [First line/paragraph concept]
- **Closing Hook**: [Cliffhanger or transition]

### 4. Writing Guidelines
Follow these principles:
- **Show, Don't Tell**: Use sensory details and action
- **Voice Consistency**: Match established character voices
- **Pacing**: Balance action, dialogue, and introspection per bible
- **Hooks**: Open and close strongly to retain readers
- **Continuity**: Reference past events naturally
- **Foreshadowing**: Plant seeds for future payoffs

### 7. Supermemory Update
After completing the chapter:
1. Store chapter summary in Supermemory
2. Update character states
3. Add new world-building details
4. Track plot thread status
5. Log foreshadowing for future reference

## Quality Checklist
- [ ] Opening hook engages immediately
- [ ] Chapter advances plot/character/world
- [ ] Voice matches established characters
- [ ] Continuity maintained with previous chapters
- [ ] Pacing follows story bible guidelines
- [ ] Closing hook compels readers to continue
- [ ] No contradictions with established canon
- [ ] Foreshadowing/callbacks included where appropriate

## Output
Provide the complete chapter with all supporting notes and context updates for Supermemory.
